# Entry Assessment Task Pack

## 英文技术面试课程入学测试任务包

欢迎加入课程。为帮助导师更准确地了解你的英文表达基础、技术背景和目标岗位方向，请按要求完成以下入学测试任务，并在规定时间内提交录音与文档作业。

**Estimated Time to Complete:**
预计总用时约 **45–75 分钟**，具体时间会因个人英文基础、岗位方向和准备习惯有所不同。
一般可参考如下：

* Warm-up Interview Questions：约 10–15 分钟
* Theory-based Technical Questions：约 15–25 分钟
* Jira Story Translation Task：约 10–15 分钟
* Scenario-based E-mail / Slack Writing Task：约 10–20 分钟

**Submission Method:**
本次作业可直接在 **微信群** 内完成交付。
录音、文档作业、简历以及目标岗位 JD（如有）均可统一提交至微信群，由助教老师收集并整理。

# Part 1: Warm-up Interview Questions

## 热身面试题（录音作业）

请对以下 2 个问题分别进行英文口语回答，并提交录音。
建议每题回答 **1–2 分钟**。请尽量自然表达，不需要逐字背稿。

### Q1

**Could you please introduce yourself and briefly describe your current role or most recent role?**

### Q2

**Why are you interested in applying for overseas or remote technical roles at this stage of your career?**

# Part 3: Sample Jira Story Translation Task

## Jira Story 翻译作业（文档作业）

请将以下 Jira Story 内容翻译成自然、简洁、专业的中文。

### Sample Jira Story

**Title:** Investigate API Response Delay on User Profile Service

**Description:**
Recently, we have noticed that the response time of the user profile service has increased during peak hours. This issue may affect the user experience on the mobile application.
Please investigate the root cause of the latency increase, review recent deployment changes, and check whether any database queries or third-party API calls are contributing to the delay.
If necessary, work with the backend team to identify possible optimizations and provide an update in the team channel by the end of the day.

**Acceptance Criteria:**

* Root cause analysis is documented
* Key findings are shared with the team
* Next-step recommendations are proposed if the issue is not fully resolved
