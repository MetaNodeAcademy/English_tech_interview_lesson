# 英语面试通关课课程素材框架（8周）

> 说明：该目录为课程交付素材骨架，你可以逐步将每周课件、作业、反馈、模板填充到对应目录。

## 目录结构（按周拆分）

- `01_week1_diagnosis/`：入学测评与双师诊断
- `02_week2_foundation/`：英文自我介绍与技术背景表达搭建
- `03_week3_tech_terms/`：技术术语与技术问题表达强化
- `04_week4_behavior/`：行为面试与 STAR 回答训练
- `05_week5_project_resume/`：项目深挖、追问应对与简历优化
- `06_week6_mock_interview/`：全真模拟与复盘
- `07_week7_remote_work_english/`：远程工作英语场景训练
- `08_week8_docs_listening_graduation/`：文档阅读、听力与结营规划
- `samples/`：示例模板（可复制后按学员定制）
