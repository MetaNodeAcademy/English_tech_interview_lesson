# 英语面试通关课课程素材框架（8周）

> 说明：该目录为课程交付素材骨架，你可以逐步将每周课件、作业、反馈、模板填充到对应目录。

## 目录结构

- `00_program_overview/`：课程定位、师资、目标学员、课程形式
- `01_week1_diagnosis/`：入学测评与双师诊断
- `02_week2_3_foundation/`：英文技术表达打底
- `03_week4_5_behavior_project/`：行为面与项目输出强化
- `04_week6_mock_interview/`：全真模拟与复盘
- `05_week7_remote_work_english/`：远程工作英语场景训练
- `06_week8_docs_listening_graduation/`：文档阅读、听力与结营规划
- `07_final_deliverables/`：八周整体交付成果汇总
- `08_advanced_services/`：后续增值服务模块
- `samples/`：示例模板（可复制后按学员定制）
