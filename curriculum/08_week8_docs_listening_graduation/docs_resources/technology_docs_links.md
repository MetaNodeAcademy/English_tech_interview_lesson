## G. Week 8 训练任务建议（可复制）
1. 每天选 1 篇官方文档，输出 5 句英文 summary。
2. 从文档提取 10 个高频术语，写中英对照 + 1 句例句。
3. 每 2 天做一次 3 分钟英文口头复述（recording）。
4. 每周做 1 次“文档 → Jira/Slack 更新”转换练习。
