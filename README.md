# English Tech Interview Lesson

一个面向程序员的 **8 周技术英语面试训练课程仓库**，聚焦：
- 技术自我介绍与背景表达
- 高频 IT 术语与项目表达
- STAR 行为面试回答
- Mock Interview 与复盘改进
- 远程协作英语与技术文档阅读

## 学习路径（8 周）

1. **Week 1: Diagnosis**  
   语言能力评估、入学任务、初始画像建立。
2. **Week 2: Foundation**  
   打磨技术背景介绍与自我介绍模板。
3. **Week 3: Tech Terms**  
   强化高频 IT 术语及例句表达。
4. **Week 4: Behavior**  
   使用 STAR 框架组织行为面试答案。
5. **Week 5: Project & Resume**  
   项目深挖问答与简历表达优化。
6. **Week 6: Mock Interview**  
   进行模拟面试、记录反馈并迭代。
7. **Week 7: Remote Work English**  
   练习远程协作场景下的英文沟通。
8. **Week 8: Docs, Listening & Graduation**  
   技术文档阅读、听力训练与毕业计划。
